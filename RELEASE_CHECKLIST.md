# Public release checklist

- [ ] LSU/RAISE LAB approves the open-source license and copyright line.
- [ ] Repository URL is final in `README.md`, `pyproject.toml`, and `CITATION.cff`.
- [ ] All author names are correct.
- [ ] No secrets, proprietary data, CPLEX binaries, or reviewer material are present.
- [ ] `pytest` passes.
- [ ] `examples/quickstart.py` runs in a clean environment.
- [ ] The full example runs with the documented Qiskit versions.
- [ ] README limitations match the implementation.
- [ ] GitHub Actions passes.
- [ ] Release tag `v1.0.0` is created.
- [ ] Paper citation and DOI are added after publication.
